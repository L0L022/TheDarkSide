C’est quoi ?

Atom est un éditeur de texte multi-plateforme développé par Gihub. Son plus grand atout est qu’il est hautement configurable et personnalisable grâce à un puissant système de paquets (extensions). Atom n’est donc pas livré seul, une multitude de paquets s’y ajoute pour rendre son utilisation plus confortable et productive.

Comment on l’utilise ?

Un raccourci Atom est normalement disponible sur le bureau. Son démarrage est un peu long. Pour ouvrir un fichier ou un dossier ça se passe dans le menu : File → Open File ou Open Folder ou encore Add Project Folder (mon préféré). Pour configurer Atom, que ce soit son comportement, son thème ou un paquet : Edit → Preferences. Pour ouvrir le manuel en ligne : Help → Documentation. Toutes les actions liées aux paquets sont dans le menu : Packages.
